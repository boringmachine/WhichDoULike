WhichDoULike
=============



概要
-------------
WhichDoULikeは、imgディレクトリに保存された画像ファイルをランダムに2つ取得し、その2つのファイルのどちらが好きなのかを、ユーザに選択させます。  
  
ユーザが画像をクリックした回数は、クリックした画像パスをキーとした連想配列形式(JSON)で保存されます。

動作環境
-------------
HTML5対応ブラウザ  
PHP5.2以上  
Apache HTTP Server 2.2
